A bookmarklet is a bookmark that contains a javascript function as the URL. The function runs when clicked on.


See here if you still don't know what a bookmarklet it, or just take my word for it.

If you don't know how to add these refer to this video clip.


Vid is outdated but the concept is to copy the function (should always start with "javascript:(function()" ). Copy to bookmarks bar, and rename to whatever you want. Then whenever you click the bookmark it'll run the function.
